#### 基于React的移动端地区选择器组件

##### Installation

npm 
> npm install react-area-selector


##### Example

```js
<AreaSelector
    itemHeight={34}
    rowCount={7}
    show={true}
    onChange={this.changeHandle}
    onCancel={this.cancelHandle}
    defaultText={['陕西省', '西安市', '雁塔区']}/>

```
